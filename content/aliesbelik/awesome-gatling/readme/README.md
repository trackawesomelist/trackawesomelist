*   [gatling-sbt-plugin (‚≠ê107)](https://github.com/gatling/gatling-sbt-plugin) - Gatling SBT plugin to integrate Gatling with SBT, allowing to use Gatling as a testing framework.

*   [gatling-junitrunner (‚≠ê3)](https://github.com/Pravoru/gatling-junitrunner) - JUnit wrapper around Gatling simulations.
*   [gatling-grpc (‚≠ê127)](https://github.com/phiSgr/gatling-grpc) - Gatling load test plugin for gRPC.

*   [gatling-mqtt (‚≠ê43)](https://github.com/mnogu/gatling-mqtt) - A Gatling plugin for stress testing MQTT.
*   [gatling-kafka (‚≠ê67)](https://github.com/mnogu/gatling-kafka) - A Gatling plugin for stress testing Apache Kafka protocol.
*   [gatling-kafka (‚≠ê13)](https://github.com/Amerousful/gatling-kafka) - Gatling plugin for Kafka.

*   [gatling-amqp-plugin (‚≠ê5)](https://github.com/galax-io/gatling-amqp-plugin) - Plugin for support performance testing with AMQP in Gatling (3.2.x).

*   [gatling-thrift (‚≠ê18)](https://github.com/3tty0n/gatling-thrift) - Gatling third party plugin for Apache Thrift.
*   [gatling-bolt (‚≠ê2)](https://github.com/sarmbruster/gatling-bolt) - Support Neo4j Bolt protocol for Gatling.
*   [gatling-zeromq (‚≠ê5)](https://github.com/softwaremill/gatling-zeromq) - A Gatling stress test plugin for ZeroMQ protocol.
*   [gatling-dubbo (‚≠ê151)](https://github.com/youzan/gatling-dubbo) - A Gatling plugin for running load tests on Apache Dubbo.
*   [gatling-wait (‚≠ê7)](https://github.com/Amerousful/gatling-wait) - Plugin that simplifies waiting for specific events allowing customizable conditions, attempt management, and error handling.
