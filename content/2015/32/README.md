## Saving current state of tracked files without commiting

```sh
git stash
```

**Alternatives:**

```sh
git stash push
```
## Saving current state including untracked files

```sh
git stash -u
```

**Alternatives:**

```sh
git stash push -u
```

git stash push --include-untracked
