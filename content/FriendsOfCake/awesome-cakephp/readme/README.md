## Authentication and Authorization

*Plugins and libraries for implementing authentication and authorization.*

*   [ADmad/SocialAuth plugin (‚≠ê51)](https://github.com/ADmad/cakephp-social-auth) - A plugin which allows you to authenticate using social providers like Facebook/Google/Twitter etc. using [SocialConnect/auth (‚≠ê554)](https://github.com/SocialConnect/auth) social sign on library.
*   [ApiTokenAuthenticator plugin (‚≠ê3)](https://github.com/rrd108/api-token-authenticator) - A simple token authentication plugin for CakePHP REST APIs.
*   [Authentication plugin (‚≠ê116)](https://github.com/cakephp/authentication) - Official CakePHP authentication middleware plugin.
*   [Authorization plugin (‚≠ê75)](https://github.com/cakephp/authorization) - Official CakePHP authorization stack.
*   [CakeDC/Users plugin (‚≠ê525)](https://github.com/CakeDC/users) - Complete user management (admin panel, remember me, etc), Social login (FB, Twitter, LinkedIn, Google, Instagram), RBAC, API and more.
*   [TinyAuth plugin (‚≠ê129)](https://github.com/dereuromark/cakephp-tinyauth) - Authentication and role-based (single/multi) authorization as very light-weight approach.
*   [Tools:Passwordable (‚≠ê335)](https://github.com/dereuromark/cakephp-tools) - Containing [Passwordable behavior (‚≠ê335)](https://github.com/dereuromark/cakephp-tools/blob/master/docs/Behavior/Passwordable.md) for a DRY approach on password hashing.
*   [TwoFactorAuth plugin (‚≠ê37)](https://github.com/andrej-griniuk/cakephp-two-factor-auth) - Allows two factor authentication using Google Authenticator or similar app to generate one-time codes. Based on [RobThree/TwoFactorAuth (‚≠ê1.1k)](https://github.com/RobThree/TwoFactorAuth) library.

*   [TestHelper plugin (‚≠ê7)](https://github.com/dereuromark/cakephp-test-helper) - Provides testing enhancements and TDD support as browser backend.
