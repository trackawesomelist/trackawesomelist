## Lighting

**[`^        back to top        ^`](#Contents)**

### Lighting Software

*   [Avolites Titan](https://www.avolites.com/software/downloads/titan-pc-suite) - PC version of the Titan software which runs on the Avolites desk. Free to use until version 11.4 although it spits out random data once in a while. Requires a dongle from version 12. `‚ö† not free`.
*   [Chamsys MagicQ](https://chamsyslighting.com/) - MagiQ is a professional lighting software providing up to 64 Artnet universes for free. It includes powerful features like 2D/3D pixel mapping, effect generator, media server and visualizer.
*   [Daslight](https://www.daslight.com/) - DMX lighting software for PC and Mac.
*   [Dot2 software](https://www.malighting.com/downloads/products/dot2/) - Free, offline software for the Dot2 consoles. Includes Dot2 onPC and Dot2 3D.
*   [EOS Family](https://www.etcconnect.com/Products/Consoles/Eos-Family/) - Software for ETC's EOS-family consoles. `‚ö† freemium`.
*   [Freestyler](http://www.freestylerdmx.be/) - Free lightning control software
*   [GrandMA2 software](https://www.malighting.com/downloads/products/grandma2/) - Free, offline software for the GrandMA2 lighting console. Includes GrandMA2 OnPC and MA3D. Does not output DMX for free. `‚ö† freemium`.
*   [GrandMA3 software](https://www.malighting.com/downloads/products/grandma3/) - Free, offline software for the **new** GrandMA3 lighting console. Includes GrandMA3 OnPC. Does not output DMX for free. `‚ö† freemium`.
*   [Hog 4 software](https://www.highend.com/products/consoles/) - Free, offline software for the Hog 4 lighting console. Includes Hog 4 pc. Does not output DMX for free. `‚ö† freemium`.
*   [LightFactory](https://www.lightfactory.net/) - PC Lighting Control software witch also is the backend of the NEO Console. `‚ö† not free`.
*   [ONYX](https://obsidiancontrol.com/onyx) - Modern lighting control system with alot of features.
*   [QLC+](https://www.qlcplus.org/) - QLC+ is a free and cross-platform software to control DMX or analog lighting systems like moving heads, dimmers, scanners etc. [Source Code (‚≠ê597)](https://github.com/mcallegari/qlcplus) `‚úì open-source`.
*   [xLights](https://xlights.org/) - Meant for home Christmas light shows but is a great way to sync lights to music on a budget. [Source Code (‚≠ê343)](https://github.com/smeighan/xLights) `‚úì open-source`.

### Visualizers

*   [Capture](https://www.capture.se/) - Documentation and visualisation software. Capture is a native macOS and Windows application that lets you work with lighting, video, laser, moving scenery and water effects. Does have a free Student Edition. `‚ö† freemium`.
*   [Depence¬≤](https://www.syncronorm.com/products/depence2/overview/) - Depence¬≤ is a unique application, which combines the design and visualization process with the potential to control the real show. `‚ö† not free`.
*   [Ma3D](https://www.malighting.com/downloads/products/grandma2/) - **Free** lighting visualizer for GrandMA consoles/software.
*   [wysiwyg](https://cast-soft.com/wysiwyg-lighting-design/) - wysiwyg is an all-in-one lighting design software with fully integrated CAD, plots, data, visualization and virtual show control. `‚ö† not free`.

### Audio Software

*   [Audacity](https://www.audacityteam.org/) - Audio file editor and analysis tool, great for cropping down sound effects and music tracks. `‚úì open-source`.
*   [Ardour](https://ardour.org/) - A full realtime audio recording and editing suite. [Source Code (‚≠ê2.7k)](https://github.com/Ardour/ardour) `‚úì open-source`.
*   [da-Share MultiPlay](https://www.da-share.com/software/multiplay/) - MultiPlay is a Windows based program designed to play audio cues for theatre or corporate use. It is free to use.
*   [Linux Show Player](https://www.linux-show-player.org/) - Free cue player designed for stage productions. [Source Code (‚≠ê147)](https://github.com/FrancescoCeruti/linux-show-player) `‚úì open-source`.
*   [QLab](https://qlab.app/) - QLab is sound, video, and lighting control for macOS. `‚ö† freemium`.
*   [REAPER](https://www.reaper.fm/) - REAPER is a complete digital audio production application for computers, offering a full multitrack audio and MIDI recording, editing, processing, mixing and mastering toolset. `‚ö† freemium`.

### Audio tools

*   [ARTA](http://www.artalabs.hr/) - A collection of programs for audio measurements and analysis.
*   [REW](https://www.roomeqwizard.com/) - REW is free room acoustics analysis software for measuring and analysing room and loudspeaker responses. Free SMAART alternative.

## Tools

**[`^        back to top        ^`](#Contents)**

*   [Livescript (‚≠ê4)](https://github.com/Netlob/livescript) - Insert a musical/theatre-script from Google Docs and use this for a live "autocue" and scroller with everyone on the site. `‚úì open-source`.

### CADs

*   [AutoCAD](https://www.autodesk.com/products/autocad) - AutoCAD¬Æ is computer-aided design (CAD) software that architects, engineers, and construction professionals rely on to create precise 2D and 3D drawings. `‚ö† not free`.

### Timecode

Timecode is a sequence of numeric codes generated at regular intervals by a timing synchronization system. Timecode is used in video production, show control and other applications which require temporal coordination or logging of recording or actions.

*   [El-Tee-See](http://elteesee.pehrhovey.net/) - LTC wav file generator.
*   [TCgenerator (‚≠ê3)](https://github.com/dimitriCGNL/TCGenerator) - Reaper time marker to GrandMA2 TC Generator. `‚úì open-source`.
*   [Timecode (‚≠ê30)](https://github.com/MrExplode/Timecode) - ArtNet and LTC timecode generator, made with Java. `‚úì open-source`.
*   [Time MIDI sender (‚≠ê0)](https://github.com/TheGreyDiamond/Time-MIDI-sender) - A free tool for sending time data to DasLight (via MIDI). `‚úì open-source`.
*   [TimelordMTC](https://timelord-mtc.com/) - Advanced timecode management tool. `‚ö† not free`.
*   [M2Q (‚≠ê4)](https://github.com/lorenzofattori/M2Q) - Python application capable to trigger Chamsys Consoles/Software via MIDI for advanced synchronized shows. `‚úì open-source`.

### Monitoring

*   [AES67 WebMeter (‚≠ê6)](https://github.com/zjstraus/AES67-WebMeter) - Server that receives an AES67 audio stream and presents a webpage with meters. `‚úì open-source`.
*   [meters.lv2 (‚≠ê143)](https://github.com/x42/meters.lv2) - meters.lv2 is a collection of audio-level meters with GUI in LV2 plugin format. `‚úì open-source`.
*   [Open Sound Meter](https://opensoundmeter.com/) - Clean, open-source sound meter. Supports RTA, magnitude, phase and impulse response, coherence, group delay and more. `‚úì open-source`.
