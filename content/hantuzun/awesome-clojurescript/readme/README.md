## Resources

### Books

*   [ClojureScript Unraveled](https://leanpub.com/clojurescript-unraveled) ‚Äì An open source book about the ClojureScript language that covers all the language features, how to use the compiler and the tooling for building applications and libraries.
*   [ClojureScript: Up and Running](https://shop.oreilly.com/product/0636920025139.do) ‚Äì An introduction to ClojureScript written by big guns of Clojure Stuart Sierra and Luke VanderHart.
*   [Clojure, The Essential Reference](https://www.manning.com/books/clojure-the-essential-reference) ‚Äì A reference book on the Clojure language and standard library.
*   [Etudes for ClojureScript](https://shop.oreilly.com/product/0636920043584.do) ‚Äì A hands-on-book of 30 companion exercises or √©tudes for introducing ClojureScript.
*   [Learning ClojureScript](https://www.packtpub.com/web-development/learning-clojurescript) ‚Äì Master the art of agile single page web application development with ClojureScript.
*   [Transforming Data with ClojureScript](https://langintro.com/cljsbook) ‚Äì A beginner's guide to ClojureScript with interactive examples and exercises, "focusing on the main task that programming does‚Äîtransforming data".

### Courses

*   [ClojureScript Koans](http://clojurescriptkoans.com/) ‚Äì A great way to start learning Clojure and ClojureScript on the web with interactive challanges.
*   [Lambda Island](https://lambdaisland.com/) ‚Äì A full-stack web development course including front-end with ClojureScript, back-end with Clojure, language essentials, security, internet standards and system administration.
*   [Learn Reagent](https://www.learnreagent.com/) ‚Äì An introduction to ClojureScript with Reagent for developers. 1 hour free content is available. [learn re-frame](https://www.learnreframe.com/) is its follow up.
*   [Learning ClojureScript](https://purelyfunctional.tv/) ‚Äì A 2h 11m long introduction to Clojurescript by O'Really Online Learning. Also available on [Udemy](https://www.udemy.com/course/learning-clojurescript/).

### Videos

*   [ClojureScript for Skeptics](https://www.youtube.com/watch?v=gsffg5xxFQI) ‚Äì Derek Slager's talk for Clojure Conj 2015 where he is Discussing the many reasons why ClojureScript is in fact a very pragmatic language to consider for web development.
*   [Interactive programming Flappy Bird in ClojureScript](https://www.youtube.com/watch?v=KZjFVdU8VLI) ‚Äì A 5 minutes video from 2014 demostrating the power of ClojureScript's developer experience.

## Awesome ClojureScript

### Canvas

*   [Monet (‚≠ê165)](https://github.com/rm-hull/monet) ‚Äì A small ClojureScript library to make it easier to work with canvas and visuals.
*   [Quamolit (‚≠ê2)](https://github.com/Quamolit/quamolit) ‚Äì A tiny declarative animation library , inspired by React.

### Client/Server Communication

*   [cljs-ajax (‚≠ê642)](https://github.com/JulianBirch/cljs-ajax) ‚Äì A simple Ajax client for ClojureScript and Clojure.
*   [Fetch (‚≠ê206)](https://github.com/LightTable/fetch) ‚Äì A ClojureScript library that makes client/server interaction painless.

### Code Analysis

*   [kibit (‚≠ê1.7k)](https://github.com/jonase/kibit) ‚Äì Static code analyzer to find patterns of code that could be rewritten with a more idiomatic function or macro.

*   [Transit (‚≠ê317)](https://github.com/cognitect/transit-cljs) ‚Äì A data interchange format and set of libraries for conveying values between applications written in different programming languages.

### Data Visualization

*   [lein-figwheel (‚≠ê2.9k)](https://github.com/bhauman/lein-figwheel) ‚Äì Leiningen plugin that pushes ClojureScript code changes to the client.

*   [Truss (‚≠ê267)](https://github.com/ptaoussanis/truss) ‚Äì An opinionated assertions API for Clojure/ClojureScript.
