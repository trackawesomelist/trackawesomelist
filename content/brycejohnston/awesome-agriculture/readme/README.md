## Automation and Robotics

*   [Ant Robotics](https://antrobotics.de/) development from Ecoterra bot
*   [Acorn Rover](https://github.com/Twisted-Fields) precision farming rover, Odrive, Python.
*   [Earth Rover](https://github.com/earthrover) Ag AGV ROS1) precision farming rover,
*   [EcoTerra Bot](https://ecoterrabot.com/) Delta & Rover
*   [FarmBot](https://github.com/farmbot) -  Open source precision gardening project.
*   [Fields2Cover (‚≠ê49)](https://github.com/Fields2Cover/Fields2Cover) - Robust and efficient coverage paths for autonomous agricultural vehicles.
*   [Romi project](https://media.romi-project.eu/documents/index.html) Europe-funded research project
*   [ROS Agriculture](http://rosagriculture.org/) - Open Source community focusing on using Robot Operating System to empower farmers with robotics tools.
*   [Weedinator](https://hackaday.io/project/53896-weedinator-2019) Line following /weeding robot

## Calculators

*   [farm-calculators (‚≠ê12)](https://github.com/brycejohnston/farm-calculators) - WordPress plugin for calculating various farming and crop related information.

## Climate, Environment and Weather

*   [agroclimatology (‚≠ê15)](https://github.com/brycejohnston/agroclimatology) - Ruby client for interacting with the NASA (POWER) Agroclimatology Web Resource.
*   [bomrang (‚≠ê101)](https://github.com/ropensci/bomrang) - Australian Government Bureau of Meteorology Data from R.
*   [evapotranspiration (‚≠ê11)](https://github.com/brycejohnston/evapotranspiration) - Ruby library for calculating reference crop evapotranspiration (ETo).
*   [frostline (‚≠ê115)](https://github.com/waldoj/frostline) - A dataset, API, and python parser for USDA plant hardiness zones.
*   [GSODR (‚≠ê80)](https://github.com/ropensci/GSODR) - Global summary daily weather data in R.
*   [iem (‚≠ê109)](https://github.com/akrherz/iem) - Code that makes the Iowa Environmental Mesonet run.
*   [PyETo (‚≠ê114)](https://github.com/woodcrafty/PyETo) - Python package for calculating reference/potential evapotranspiration (ETo).
*   [pyTSEB (‚≠ê96)](https://github.com/hectornieto/pyTSEB) - A python two source energy balance model for estimation of evapotranspiration with remote sensing data.
*   [soilDB (‚≠ê60)](https://github.com/ncss-tech/soilDB) - R library for simplified access to NCSS soil databases.

## Crop Modeling, Phenotyping, and Pathology

*   [Open Plant Pathology](https://www.openplantpathology.org/) - A community that values open data and computational tools for advancing epidemiology and pathogen population biology and ecology.

## Datasets,

*   [Growstuff](https://www.growstuff.org/crops) Record keeping & crop database, nice API
*   [CWFID (‚≠ê120)](https://github.com/cwfid/dataset) - Dataset comprising field images, vegetation segmentation masks and crop/weed plant type annotations.
*   [TERRA REF](https://terraref.org) -> 1PB public domain high resolution sensor data from sorghum breeding trials ([data publication](https://doi.org/10.5061/dryad.4b8gtht99) with large files available on [globus.org at ncsa#terra-public](https://app.globus.org/file-manager?origin_id=e8feaff4-96cd-11ea-bf90-0e6cccbb0103\&origin_path=%2F))

*   [farmOS (‚≠ê550)](https://github.com/farmOS/farmOS) - Web-based farm record keeping application built with Drupal and PHP.
