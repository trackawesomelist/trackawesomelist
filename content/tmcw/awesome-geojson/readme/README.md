# awesome geojson [![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome)

GeoJSON utilities that will make your life easier.

<a href='https://www.placemark.io/'><i><img src='.github/placemark.png' width='16' alt='Placemark' height='16' /> Sponsored by Placemark, a collaborative editor for maps with great support for GeoJSON</i></a>

### operations

*   [CoGran (‚≠ê103)](https://github.com/berlinermorgenpost/cogran): Resample statistics from one set of GeoJSON polygons to another
*   [geo\_assigner (‚≠ê2)](https://github.com/stadt-karlsruhe/geo_assigner): Copy a property to all intersecting features
*   [geojson-coords (‚≠ê28)](https://github.com/mapbox/geojson-coords): Extract coordinates from GeoJSON
*   [geojson-extent](https://www.npmjs.com/package/geojson-extent): compute the bounding box of geojson features
*   [geojson-flatten (‚≠ê49)](https://github.com/tmcw/geojson-flatten): flatten multi geometries into normal geometries
*   [geojson-multiply (‚≠ê2)](https://github.com/haoliangyu/geojson-multiply):  merge normal geojson features into one multi geometry type feature
*   [geojson-js-utils (‚≠ê400)](https://github.com/maxogden/geojson-js-utils): JavaScript helper functions for manipulating GeoJSON
*   [geojson-merge (‚≠ê219)](https://github.com/mapbox/geojson-merge): Merge multiple GeoJSON files into one FeatureCollection.
*   [geojson-normalize (‚≠ê30)](https://github.com/mapbox/geojson-normalize): normalize any geojson object into a geojson featurecollection
*   [geojson-pick](https://www.npmjs.com/package/geojson-pick): remove all but specified properties from features in a geojson featurecollection
*   [geojson-random (‚≠ê72)](https://github.com/tmcw/geojson-random): generate random geojson points, lines, and polygons
*   [geojson-rewind (‚≠ê109)](https://github.com/mapbox/geojson-rewind): enforce ring winding order
*   [rfc7946-to-d3 (‚≠ê10)](https://github.com/tyrasd/rfc7946-to-d3): Converts polygon winding order between rfc7946 and [d3 compatible (‚≠ê904)](https://github.com/d3/d3-geo#d3-geo) conventions.
*   [geojson-summary (‚≠ê32)](https://github.com/mapbox/geojson-summary): get a plain-english summary of what's in a geojson file
*   [point-grouper](https://github.com/substack/point-grouper): group geojson points into containing polygons
*   [geojson-join (‚≠ê35)](https://github.com/tmcw/geojson-join): join geojson against json, dbf, and csv files
*   [simplify-geojson (‚≠ê162)](https://github.com/maxogden/simplify-geojson): apply the ramer-douglas-peucker line simplification to geojson features or feature collections in JS or on the CLI
*   [turf (‚≠ê7.7k)](https://github.com/Turfjs/turf): collection of functions for spatial operations and analysi
*   [turf\_dart (‚≠ê29)](https://github.com/dartclub/turf_dart): a turf.js port for Dart and Flutter
*   [de9im (‚≠ê22)](https://github.com/dpmcmlxxvi/de9im): collection of fast functions for spatial javascript operations
*   [winnow (‚≠ê89)](https://github.com/dmfenton/winnow): run sql queries against geojson with javascript
*   [geojson-id-streaming (‚≠ê2)](https://github.com/andrewharvey/geojson-id-streaming): Add a unique id to each Feature
*   [geojson-mesh (‚≠ê12)](https://github.com/andrewharvey/geojson-mesh): extract shared borders of Polygons in a GeoJSON
*   [geojson-polygon-labels (‚≠ê73)](https://github.com/andrewharvey/geojson-polygon-labels): generate point labels from GeoJSON polygons
*   [geojsontools (‚≠ê18)](https://github.com/micolous/geojsontools): Python tools for manipulating GeoJSON files
*   [geojson-indoor-tools (‚≠ê0)](https://github.com/armanjr/geojson-indoor-tools): Python tools for editing indoor GeoJSON features: adding room heights, building walls and creating doors

### editors & viewers

*   [geojson.io](http://geojson.io/): web-based editor, supports many filetype imports & exports, operations, sharing via GitHub
*   [umap](http://umap.openstreetmap.fr/en/): web-based editor, supports sharing on-site
*   [geojson.tools](http://geojson.tools/): web-based editor from HERE XYZ
*   [mapstarter](http://mapstarter.com/): helps generate svg, images, and code from GeoJSON
*   [gjv (‚≠ê75)](https://github.com/anandthakker/gjv): electron app enables viewing GeoJSON locally and hack on it with turf; works offline
*   [GeoJSON editor](https://tomscholz.github.io/geojson-editor/): a modified version of Google's simple geojson editor
*   [geojson2image (‚≠ê11)](https://github.com/brycejohnston/geojson2image): library for generating images from GeoJSON
*   [dropchop](http://dropchop.io/): browser-based GIS based on Turf.js
*   [Web GIS](https://drewweth.github.io/geodebugger): tools to view, inspect, and manipulate geospatial data
*   [img2geojson](https://caseymm.github.io/img2geojson/): digitize images by overlaying on a web map and drawing

### validation

*   [geojsonhint (‚≠ê247)](https://github.com/mapbox/geojsonhint): find errors in your geojson files
*   [check-geojson (‚≠ê46)](https://github.com/placemark/check-geojson): alternative to geojsonhint
*   [geojson-validation](https://www.npmjs.com/package/geojson-validation): many validation functions, plus simple gjv command line tool
*   [gjf (‚≠ê91)](https://github.com/yazeed44/gjf): A tool to fix invalid GeoJSON objects
*   [Jest-GeoJSON (‚≠ê6)](https://github.com/M-Scott-Lassiter/jest-geojson): an extended set of matcher functions for the Jest Javascript unit testing framework tailored to evaluating GeoJSON data

### services

*   [geojsonio-cli (‚≠ê158)](https://github.com/mapbox/geojsonio-cli): send geojson features to geojson.io from your command line
*   [geojsonio-extension (‚≠ê1)](https://github.com/mapbox/geojsonio-extension): chrome extension for editing github files in geojson.io
*   [geojsonlint](http://geojsonlint.com/): REST interface for GeoJSON validation
*   [mapshaper](http://mapshaper.org/): Simple interface for simplification and conversion of GeoJSON and TopoJSON
*   [koop](https://koopjs.github.io): Server with plugins that recast Esri, GitHub, Socrata and other services as GeoJSON endpoints
*   [featureserver (‚≠ê95)](https://github.com/featureserver/featureserver): An open source Esri-Style Feature Server

### conversion

*   **CSV**
    *   [csv2geojson (‚≠ê341)](https://github.com/mapbox/csv2geojson): convert CSV to geojson
    *   [geojson2dsv (‚≠ê8)](https://github.com/tmcw/geojson2dsv): convert geojson to CSV and TSV
*   [geojson-mapnikify (‚≠ê38)](https://github.com/mapbox/geojson-mapnikify): Transform GeoJSON objects into Mapnik XML stylesheets with embedded GeoJSON data and simplestyle-spec-derived styles.
*   [geojson-vt (‚≠ê1.6k)](https://github.com/mapbox/geojson-vt): Slice GeoJSON into vector tiles on the fly in the browser
*   [vt-geojson (‚≠ê58)](https://github.com/developmentseed/vt-geojson): Extract GeoJSON from Mapbox vector tiles
*   [geojson2rtree (‚≠ê9)](https://github.com/maxogden/geojson2rtree): generate a static rtree (using terraformer) from a set of geojson features
*   [ogr2ogr](http://www.gdal.org/ogr2ogr.html): convert anything to anything
    *   [fiona (‚≠ê988)](https://github.com/toblerity/fiona): nice python interface on top of ogr
*   [shp2json](https://github.com/substack/shp2json): convert shapefile zip archives to streaming GeoJSON
*   [togeojson (‚≠ê311)](https://github.com/tmcw/togeojson): convert GPX & KML to geojson
*   [tokml (‚≠ê165)](https://github.com/mapbox/tokml): convert GeoJSON to KML
*   [topojson (‚≠ê4.4k)](https://github.com/topojson/topojson): convert GeoJSON to & from TopoJSON, join data from CSV
*   [wellknown (‚≠ê242)](https://github.com/mapbox/wellknown): convert WKT to GeoJSON
*   [betterknown (‚≠ê32)](https://github.com/placemark/betterknown): convert WKT to GeoJSON (successor to wellknown)
*   [esri2open (‚≠ê247)](https://github.com/project-open-data/esri2open) converts proprietary Esri formats to GeoJSON
*   [gtfs2geojson (‚≠ê25)](https://github.com/tmcw/gtfs2geojson): convert GTFS transit data to GeoJSON
*   [geoxform (‚≠ê13)](https://github.com/koopjs/geoxform): convert any size stream of geojson into a streaming CSV, Shapefile or KML, etc.
*   [supercluster (‚≠ê1.6k)](https://github.com/mapbox/supercluster): A fast GeoJSON clustering library for browsers and Node.
*   **SVG**
    *   [geojson2svg (‚≠ê164)](https://github.com/gagan-bansal/geojson2svg): Converts GeoJSON to SVG string given SVG view port size and maps extent.
    *   [geojson-svgify (‚≠ê15)](https://github.com/juliuste/geojson-svgify): Convert GeoJSON geometry paths to SVG polyline elements.
    *   [geojson-to-svg-cli (‚≠ê26)](https://github.com/derhuerst/geojson-to-svg-cli): Command line tool to convert GeoJSON to SVG.
    *   [svg-to-geojson (‚≠ê107)](https://github.com/mapbox/svg-to-geojson): Drag and drop SVG on the map and it returns GeoJSON back to you.
*   [geojson2ndjson](https://www.npmjs.com/package/geojson2ndjson): convert GeoJSON to/from newline-delimited GeoJSON
*   [geobuf](https://www.npmjs.com/package/geobuf/v/0.2.1): convert GeoJSON to/from GeoBuf
*   **OpenStreetMap**
    *   [osmtogeojson (‚≠ê589)](https://github.com/tyrasd/osmtogeojson): convert OpenStreetMap data to GeoJSON
    *   [osm2geojson (‚≠ê11)](https://github.com/rclark/osm2geojson): converts OSM XML to GeoJSON
    *   [osm2geojson (‚≠ê68)](https://github.com/aspectumapp/osm2geojson): (Python) converts OSM XML and Overpass XML/JSON to GeoJSON
    *   [minjur (‚≠ê58)](https://github.com/mapbox/minjur): converts OpenStreetMap data to GeoJSON faster than anything else
    *   [geojsontoosm (‚≠ê14)](https://github.com/tyrasd/geojsontoosm): convert GeoJSON to OSM XML
    *   [geojson2osm (‚≠ê6)](https://github.com/Rub21/geojson2osm): convert GeoJSON to OSM XML
    *   [osm-and-geojson (‚≠ê89)](https://github.com/aaronlidman/osm-and-geojson): convert GeoJSON to OSM XML and OSM XML to GeoJSON
    *   [geojson2osm-es6 (‚≠ê0)](https://github.com/DenisCarriere/geojson2osm-es6/): convert GeoJSON to OSM XML
    *   [query-overpass (‚≠ê141)](https://github.com/perliedman/query-overpass) CLI to make Overpass query and transform result to GeoJSON. Based on [osmtogeojson (‚≠ê589)](https://github.com/tyrasd/osmtogeojson)
